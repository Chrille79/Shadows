# fas1-keepers

4 stil-spikade bilder från Fas 1 (genererade 2026-04-16).

**Flytta PNG:erna hit från `ComfyUI/output/`.** De är stil-facit för
resten av projektet — input till LoRA-träning i Fas 3, visuellt mål
för tile-produktion i Fas 4.

## Innehåll

- [x] `SD1.5_00005_.png` — Djungel, cyan-magenta bioluminescent variant.
      Rikare färg, hängande vines, magenta löv i förgrund, cyan atmosfärisk haze.
- [x] `SD1.5_00024_.png` — Djungel, mörk måne-variant. Dark desaturated blue-gray
      palette, silhuett-tunga träd, måne i bakgrunden, minimal färg-accent.
- [x] `SD1.5_00026_.png` — Djungel, magenta-plant variant. Mörk bas med magenta
      löv i förgrund och små varma ljuspunkter i mitten (pickup-kandidater).
- [x] `SD1.5_00030_.png` — Grotta, minimalist tre-tons. Pure black silhuetter,
      grå stenformationer, pale blue ambient haze. Ingen färg-accent.

## Återfå prompt/seed från en PNG

Dra PNG-filen på ComfyUI-canvasen. Prompt, seed och alla inställningar
rekonstrueras automatiskt från metadatan.
